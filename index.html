Mengenal HTML

HTML adalah singkatan dari HyperText Markup Language. Ini merupakan bahasa markup standar untuk pembuatan halaman web. HTML memungkinkan pembuatan dan struktur bagian, paragraf, dan tautan menggunakan elemen HTML (blok bangunan dari halaman web) seperti tag dan atribut.

Fungsi

Cara Kerja

Aturan Dasar

  • Struktur Konten Web
    HTML berfungsi sebagai bahasa markup yang digunakan untuk membentuk struktur konten di web, termasuk teks, gambar, tautan, dan media lainnya.
  • Interaktivitas dan Navigasi
    HTML memungkinkan pembuatan tautan (hyperlink) untuk navigasi antar halaman, serta formulir untuk pengguna berinteraksi dengan menyampaikan data.
  • Semantik dan Aksesibilitas
    HTML mendukung markup semantik dengan elemen-elemen seperti tag header, nav, article, yang meningkatkan aksesibilitas dan memastikan pemahaman konten oleh mesin pencari.
  • Penulisan Dokumen HTML
    Setiap halaman web dimulai dengan deklarasi dokumen HTML, yang dituliskan sebagai !DOCTYPE html. Ini memberitahu browser bahwa dokumen ini adalah dokumen HTML5.
  • Elemen HTML
    HTML menggunakan elemen-elemen untuk mendefinisikan struktur dan konten halaman web. Elemen dimulai dan diakhiri dengan tag. Contohnya, p untuk paragraf, h1 untuk heading level 1, a untuk tautan, dll
  • Struktur Dasar HTML
    Sebuah dokumen HTML biasanya memiliki struktur dasar, termasuk elemen-elemen seperti html, head, dan body
  • Deklarasi Dokumen HTML
    Setiap dokumen HTML dimulai dengan deklarasi untuk menentukan versi HTML yang digunakan.
  • Elemen HTML
    HTML menggunakan elemen untuk menentukan struktur dan konten dokumen. Setiap elemen dimulai dengan tag pembuka dan diakhiri dengan tag penutup (kecuali untuk elemen yang bersifat mandiri).
  • Struktur Dasar HTML
    Dokumen HTML memiliki struktur dasar dengan elemen html, head, dan body. Tag html membungkus seluruh dokumen, head berisi informasi terkait dokumen seperti judul, dan body berisi konten utama.

Mengenal CSS

CSS, yang merupakan singkatan dari Cascading Style Sheets, adalah bahasa stylesheet yang digunakan untuk menjelaskan tata letak dari suatu dokumen yang ditulis dalam HTML atau XML. CSS mengontrol tata letak, gaya, dan tampilan halaman web.

Fungsi

Cara Kerja

Aturan Dasar

  • Pemisahan Tata Letak dan Presentasi
    Salah satu fungsi utama CSS adalah memisahkan tata letak (HTML) dari presentasi (CSS). Ini memungkinkan pengembang untuk mengatur struktur dan konten HTML tanpa harus merubah gaya visual.
  • Pengaturan Gaya dan Penyajian Visual
    CSS memberikan kontrol terhadap elemen-elemen HTML, termasuk warna, ukuran teks, jenis font, margin, padding, dan tata letak lainnya.
  • Responsivitas dan Adaptasi Layar
    Dengan menggunakan media queries, CSS memungkinkan pengembang untuk membuat desain responsif yang dapat menyesuaikan tampilan halaman web dengan berbagai ukuran layar, seperti pada perangkat mobile, tablet, dan desktop.

Sejarah HTML
